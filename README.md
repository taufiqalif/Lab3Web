## membuat list

`HTML List memungkinkan untuk mengelompokkan sekumpulan item terkait dalam sebuah daftar.`

ada 3 jenis list:

1.  Ordered List
    menggunakan tag

        <ol>
          <li>...</li>
        </ol>

2.  Unordered List
    menggunakan tag

        <ul>
          <li>...</li>
        </ul>

3.  Description List
    menggunakan tag

        <dl>
          <dt>...</dt>
          <dd>...</dd>
        </dl>
